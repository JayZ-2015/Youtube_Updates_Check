Project Documentation

YouTube Channel
Activity Checker

Monitors YouTube channels daily and sends Slack notifications when new videos are published — fully automated via GitHub Actions.

Python 3.13 GitHub Actions Slack Webhooks No API Key Required 7 Channels EST Timestamps May 2026
01 — Overview

How It Works

The script uses YouTube's public RSS feed to check for new videos — no API key needed. GitHub Actions runs it every day automatically and sends a Slack message when new content is found. All timestamps are displayed in EST.

⏰ GitHub Actions 9am UTC daily
→
📡 RSS Feed YouTube channels
→
🔍 Check Videos Last 24 hours
→
💬 Slack Alert Table + links

06 — Scheduler

GitHub Actions Schedule

Runs automatically every day at 9:00 AM UTC (5:00 AM EST). Can also be triggered manually from the Actions tab on GitHub. All times in notifications are displayed in EST.

on:
  schedule:
    - cron: '0 9 * * *'   # 9am UTC daily
  workflow_dispatch:        # manual trigger

To change the time, edit .github/workflows/daily_check.yml and update the cron expression:

CronMeaning
0 9 * * *Every day at 9:00 AM UTC (5:00 AM EST)
0 14 * * *Every day at 2:00 PM UTC (10:00 AM EST)
0 0 * * *Every day at midnight UTC (8:00 PM EST)
